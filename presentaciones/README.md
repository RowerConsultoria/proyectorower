# Presentaciones

Presentaciones web del Proyecto Rower (Junta Directiva, comités, sesiones de trabajo).

## Convención

- Una carpeta por presentación: `presentaciones/<fecha>-<tema>/` (ej. `2026-07-24-junta-directiva/`), con un `index.html` autocontenido.
- Usar la paleta y tipografía institucional definidas en [`gestion/CONVENCIONES.md`](../gestion/CONVENCIONES.md).
- Enlazar cada presentación nueva desde el portal (`index.html` en la raíz).
